This is menu in one of dental apps.

ОСНОВНОЕ МЕНЮ (просто название группы элементов в меню)

Расписание (тут очевидно, что расписание и возможность записать пациентов)
Карта пациента (открывает напрямую карточку последнего пациента)
Список пациентов (список пациентов клиники)
Онлайн-запись (окно со списком пациентов, которые записались в клинику через внешние интеграции)
Обмен данными (окно с логами отправки данных через внешние интеграции)
Реестр счетов (список сформированных и оплаченных счетов, для анализа по определенным условиям)
Прайс-лист (прайс лист клиники, который также можно изменить и прочие действия с прайсом)
Прайс-лист тех. Лабораторий (тоже саоме что выше, но данные относятся к техническим лабораториям)
Документы в базе (окно в которое падают документы для отправки через внешние интеграции)
Улучшение снимков (услуга по улучшению качества снимков пациентов)
Инструкции к программе (ссылка на сайт с базой знаний по программе)

ПАРТНЕРЫ (просто название группы элементов в меню)

Юридические лица (список контрагентов, работающих с клиникой)
Реестр отправки в ЕГИСЗ (список отправок данных о посещении клинки в ЕГИСЗ)

МАРКЕТИНГ (просто название группы элементов в меню)

Реклама (окно с возможностью добавить рекламные акции, плюс возможность их анализа и вся история этих акций)
Журнал обзвона (история связи менеджеров с пациентами)
программа лояльности (бонусные и реферальные программы лояльности для пациентов)
Статистика (Аналитика по клинике и врачам, кол-во пациентов, услуг, посещений и тп)
Аналитика CallTouch (в целом для нас не самый интересный пункт)

ЗАДАЧИ И УВЕДОМЛЕНИЯ (просто название группы элементов в меню)

Журнал задач (пункт меню который позволяет назначать задачи другим врачам)
SMS и e-mail рассылка (на говорит само за себя)
Лист ожидания (список в который можно внести пациентов на будущие посещения)

ОТЧЕТЫ (просто название группы элементов в меню)

ИДС, Документы (список документов для печати на подпись пациентов)
Ведомости (окно с различными ведомостями по расходам, доходам, зарплатам врачей, оплатам Лабораторий и прочие отчетности)
Список отчетов (тут у них еще один раздел с отчетами и ведостями)
Список оплат (почему-то еще один отдельный пункт с отчетом по оплатам за  указанный период)
Дополнительные отчеты (еще один раздел с отчетами)
Реестр счетов по ОМС, ДМС (Реестр счетов в котором отражаются оплаты по платному или бесплатному мед. страхованию)

ШАБЛОНЫ (просто название группы элементов в меню)

Диагноз (шаблоны диагнозов)
Дневники (шаблоны дневников)
Первичный осмотр (шаблоны с данными по осмотру павциента)
План лечения (шаблоны планов лечения)

АДМИНИСТРИРОВАНИЕ (просто название группы элементов в меню)

Параметры пользователей (настройка прав пользователей и прочее их администрирование )
Настройка (все настройки программы)
Интеграция сервисов (настройка интеграций с внешними ресурсами)
