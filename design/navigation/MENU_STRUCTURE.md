## Структура по ролям

### 1. Владелец клиники (Owner)

**Доступ ко всему + управление:**

```
✅ Главная (полный Dashboard с финансами)
✅ Расписание (просмотр всех врачей)
✅ Задачи (все задачи клиники, назначение задач любым сотрудникам)
✅ Пациенты
✅ Шаблоны (создание и редактирование общих и личных шаблонов)
✅ Медкарта
✅ Финансы (все подразделы)
✅ Склад
✅ Аналитика (полная)
✅ Отчёты (все ведомости и реестры)
✅ Настройки (полный доступ):
   • Клиника
   • Филиалы
   • Сотрудники
   • Подписка и оплата (только Owner)
   • Дополнительные возможности (только Owner)
   • Уведомления
   • Услуги
   • Виджет онлайн-записи
   • Документы и печать
   • Интеграции
```

### 2. Врач (Doctor)

**Клиническая работа:**

```
✅ Главная (упрощённый Dashboard)
✅ Расписание (своё расписание + просмотр коллег)
✅ Задачи (мои задачи, создание задач себе и коллегам)
✅ Пациенты (все пациенты клиники)
✅ Шаблоны (создание личных шаблонов, использование общих)
✅ Зубная формула
✅ Медкарты (создание/редактирование)
✅ Услуги (просмотр прайса)
❌ Счета (только просмотр, без создания)
❌ Финансы (ограниченный доступ)
❌ Склад (только просмотр остатков)
❌ Аналитика (базовая: свои записи)
❌ Отчёты (нет доступа)
✅ Настройки (только личный профиль):
   ❌ Подписка и оплата (нет доступа)
   ❌ Дополнительные возможности (нет доступа)
   ✅ Уведомления (свои настройки)
```

### 3. Администратор (Admin)

**Операционное управление:**

```
✅ Главная (Dashboard с метриками)
✅ Расписание (управление записями всех врачей)
✅ Задачи (мои задачи, создание задач сотрудникам)
✅ Пациенты (полный доступ)
✅ Шаблоны (просмотр и использование общих шаблонов)
✅ Медкарты (просмотр)
✅ Счета (создание, отправка)
✅ Оплаты (приём платежей)
✅ Баланс пациентов
✅ Бонусы и акции
✅ Склад (управление)
✅ Аналитика (операционная)
✅ Отчёты (финансовые и операционные ведомости)
✅ Настройки (ограниченные):
   ❌ Филиалы (нет доступа)
   ❌ Сотрудники (нет доступа)
   ❌ Подписка и оплата (нет доступа)
   ❌ Дополнительные возможности (нет доступа)
   ✅ Уведомления (свои настройки)
```

## Детальная структура разделов

### 🏠 Главная (Dashboard)

**Подразделы:**
- Обзор дня (KPI, записи, задачи)
- Быстрые действия

**Роли:**
- Владелец: полные метрики + финансы
- Врач: своё расписание + свои пациенты
- Админ: операционные метрики

---

### 📅 Расписание

**Подразделы:**
- Календарь (день/неделя/месяц)
- Управление расписанием врачей
- Исключения (отпуска, больничные)

**Действия:**
- Создать запись
- Отменить/перенести
- Отметить статус (пришёл/не пришёл)

---

### ✓ Задачи

**Основной функционал:**

**Список задач:**
- Мои задачи (назначенные мне)
- Созданные мной (которые я поставил другим)
- Все задачи клиники (для владельцев/админов)
- Фильтры: по статусу, приоритету, исполнителю, дедлайну, филиалу

**Создание задачи:**
- Название и описание
- Исполнитель (выбор из списка сотрудников)
- Приоритет (🔵 Низкий, 🟡 Средний, 🟠 Высокий, 🔴 Срочно)
- Дедлайн (дата и время)
- Привязка к пациенту (опционально)
- Привязка к записи в расписании (опционально)
- Привязка к счёту (опционально)
- Чек-листы внутри задачи
- Файлы и комментарии
- Наблюдатели (кто получает уведомления о задаче)

**Статусы задач:**
- 🆕 Новая (только создана)
- 🔄 В работе (исполнитель начал работу)
- ⏸️ Отложена (временно приостановлена)
- ✅ Выполнена (завершена исполнителем)
- ✔️ Закрыта (проверена создателем)
- ❌ Отменена (больше не актуальна)

**Уведомления:**
- Когда вам назначили задачу
- Когда дедлайн приближается (за 24 часа, за 1 час)
- Когда задача просрочена
- Когда задача выполнена/закрыта
- Новые комментарии в задаче
- Изменение статуса или исполнителя

**Представления:**
- Список (с сортировкой и фильтрами)
- Канбан-доска (по статусам)
- Календарь (по дедлайнам)
- По исполнителям (групповой вид)

**Примеры задач:**
- "Позвонить Иванову И.И. для подтверждения записи на 15.01"
- "Заказать пломбировочный материал Filtek (остаток критический)"
- "Подготовить договор для пациента Петровой М.С."
- "Отправить акт выполненных работ в страховую компанию"

### 📝 Шаблоны

**Клинические шаблоны для быстрого заполнения медкарт:**

**Типы шаблонов:**

**1. Первичный приём:**
- Шаблоны первичного осмотра
- Шаблоны анамнеза (жалобы, история заболевания, анамнез жизни)
- Шаблоны внешнего обследования (лицо, лимфоузлы, ВНЧС)
- Шаблоны обследования полости рта (слизистая, язык, дёсны)

**2. Диагностика:**
- Шаблоны диагнозов (по нозологиям: кариес, пульпит, периодонтит и т.д.)
- Шаблоны описания снимков:
  - Рентген (прицельный, панорамный)
  - КТ (компьютерная томография)
  - ОПТГ (ортопантомограмма)
- Шаблоны заключений и интерпретаций

**3. Планы лечения:**
- Шаблоны планов лечения по нозологиям
- Шаблоны поэтапного лечения
- Шаблоны альтернативных планов

**4. Дневники врача:**
- Шаблоны записей во время лечения
- Шаблоны процедур (по типам услуг: пломбирование, удаление, протезирование)
- Шаблоны этапов лечения

**5. Рекомендации и документы:**
- Шаблоны рекомендаций пациенту (после удаления, после имплантации и т.д.)
- Шаблоны согласий на лечение
- Шаблоны противопоказаний
- Шаблоны описания осложнений

**Функционал:**

**Управление шаблонами:**
- Создание нового шаблона
- Редактирование существующих
- Удаление шаблонов
- Дублирование шаблонов
- Категории и теги для организации
- Поиск по названию и содержимому

**Типы шаблонов по доступу:**
- **Общие шаблоны клиники** (доступны всем врачам, редактируют владельцы)
- **Личные шаблоны врача** (видны только создателю)

**Использование:**
- Избранное (часто используемые шаблоны)
- Быстрая вставка в медкарту одним кликом
- Возможность редактирования после вставки
- История использования шаблонов

**Структура шаблона:**
- Название
- Категория (первичный осмотр, диагноз, план лечения и т.д.)
- Теги
- Текст шаблона (с возможностью форматирования)
- Переменные (автозаполнение: {ФИО_пациента}, {дата}, {врач})
- Чек-листы (опционально)

**Права доступа:**

**Владелец:**
- ✅ Просмотр всех шаблонов (личных и общих)
- ✅ Создание общих шаблонов клиники
- ✅ Редактирование/удаление любых шаблонов
- ✅ Управление категориями
- ✅ Создание личных шаблонов

**Врач:**
- ✅ Просмотр общих шаблонов клиники
- ✅ Создание личных шаблонов (видны только ему)
- ✅ Редактирование/удаление своих личных шаблонов
- ❌ Редактирование общих шаблонов клиники
- ✅ Использование любых шаблонов в медкартах

**Администратор:**
- ✅ Просмотр общих шаблонов
- ✅ Использование шаблонов (если работает с медкартами)
- ❌ Создание/редактирование шаблонов

**Примеры шаблонов:**

*Шаблон "Первичный осмотр - Кариес"*:
```
Жалобы: Боль в зубе {номер_зуба} при приёме холодной/горячей пищи.
Анамнез: Беспокоит в течение {период}.
Объективно: На жевательной поверхности зуба {номер_зуба} кариозная полость средней глубины, заполненная размягчённым дентином.
Перкуссия безболезненная. Зондирование болезненное по дну.
Диагноз: Средний кариес зуба {номер_зуба}.
```

*Шаблон "Описание ОПТГ"*:
```
ОПТГ от {дата}:
- Костная ткань альвеолярных отростков без патологических изменений
- Периапикальные ткани без очагов деструкции
- Зубы {список}: пломбированы, пломбы состоятельны
- Зубы {список}: депульпированы, корневые каналы запломбированы до апекса
- Зубы {список}: отсутствуют
```

### 📊 Аналитика

**Интерактивные дашборды и визуализация:**

**Финансовая аналитика:**
- График выручки (день/неделя/месяц/год)
- Воронка продаж (консультация → план → оплата)
- Средний чек
- Динамика прибыли по филиалам
- Структура выручки по услугам (диаграмма)

**Клиническая аналитика:**
- Загрузка врачей (визуализация по дням/неделям)
- Популярные услуги (топ-10 с графиками)
- Конверсия записей (записался → пришёл → повторный визит)
- Средняя длительность лечения
- Распределение диагнозов

**Маркетинговая аналитика:**
- Источники пациентов (откуда пришли)
- LTV (пожизненная ценность пациента)
- Retention rate (удержание пациентов)
- Эффективность акций
- Воронка из листа ожидания

**Пациенты:**
- Новые vs повторные
- Пациенты без визитов (> 6 месяцев)
- География пациентов
- Возрастная структура

**Складская аналитика:**
- Расход материалов по периодам
- ABC-анализ материалов
- Скорость оборачиваемости

**Особенности:**
- Все графики интерактивные
- Фильтры по датам, филиалам, врачам
- Экспорт в Excel/PDF
- Сравнение периодов

### 📋 Отчёты (Ведомости и реестры)

**Табличные документы для печати, экспорта и проверки:**

**Финансовые отчёты:**
- **Реестр счетов** (все счета за период с фильтрами)
  - По статусу (ожидает/оплачен/частично)
  - По врачу, пациенту, услуге
  - С суммами и датами
- **Реестр оплат** (журнал всех платежей)
  - Наличные/безналичные/смешанные
  - По кассирам
  - По способу оплаты
- **Отчёт по выручке** (детальный за период)
  - По услугам
  - По врачам
  - По филиалам
- **Задолженности пациентов** (кто и сколько должен)
- **Отчёт по услугам** (какие услуги оказаны, количество, сумма)
- **Акты выполненных работ** (для страховых компаний)

**Клинические отчёты:**
- **Реестр пациентов** (полный список с контактами)
  - С фильтрами по дате регистрации
  - По врачу
  - По статусу (активный/неактивный)
- **Журнал приёмов** (все визиты за период)
  - Дата, время, врач, пациент, услуга
  - Статус приёма (состоялся/не пришёл)
- **Реестр медкарт** (список всех медкарт)
- **Статистика по диагнозам** (какие диагнозы ставились)
- **Отчёт по гарантиям** (истекающие гарантийные сроки)

**Складские отчёты:**
- **Реестр материалов** (остатки на складе)
  - С указанием минимумов
  - С выделением критических остатков
- **Отчёт по списаниям** (что и когда списали)
  - По врачам
  - По пациентам
  - По материалам
- **Инвентаризация** (сверка остатков)
- **Отчёт по поступлениям** (закупки)

**Кадровые отчёты:**
- **Загрузка врачей** (табличный с часами работы)
- **Выполненные услуги по врачам** (что сделал каждый врач)
- **Отчёт по рабочему времени**
- **KPI сотрудников**

**Маркетинговые отчёты:**
- **Источники пациентов** (откуда пришли: реклама, сайт, рекомендации)
- **Эффективность акций** (сколько привлекли пациентов)
- **Отчёт по программам лояльности** (начисления/списания бонусов)
- **Лист ожидания** (кто в листе, статус)

**Для внешних организаций:**
- **Реестр для ОМС/ДМС** (для страховых компаний)
- **Отчёты для налоговой** (выручка, услуги)
- **Реестр для лабораторий** (заказы, статусы)

**Особенности раздела:**
- Все отчёты с фильтрами (даты, филиалы, врачи, пациенты)
- Экспорт в Excel, PDF, CSV
- Печать с настройкой формата
- Шаблоны отчётов (можно сохранить фильтры)
- История формирования отчётов
- Автоматическая отправка по расписанию (опционально)

**Разделы (в sidebar или табы на странице настроек):**

- Личные данные и контакты

**Доступ:** Только владельцы (`is_owner = 1`) + делегированные пользователи

- **Текущий тариф**
  - Название тарифа (Solo / Clinic / Enterprise)
  - Стоимость (месяц/год)
  - Лимиты (пользователи, пациенты)
  - Кнопка "Изменить тариф"
- **Способы оплаты**
  - Привязанные карты (последние 4 цифры)
  - Добавить/удалить карту
  - Сделать картой по умолчанию
- **История платежей**
  - Таблица: Дата, Период, Сумма, Статус
  - Скачать счёт (PDF)
  - Кнопка "Скачать все счета"
- **Следующее списание**
  - Дата и сумма следующего автоплатежа
  - Включая стоимость базового тарифа + активных фич

**Делегирование доступа (Phase 2):**
- Владелец может выдать права `VIEW_BILLING` или `MANAGE_BILLING` другим пользователям

- `MANAGE_BILLING` - полное управление
