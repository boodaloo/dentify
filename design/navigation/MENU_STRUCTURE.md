## Структура по ролям

### 1. Владелец клиники (Owner)

**Доступ ко всему + управление:**

```
✅ Главная (полный Dashboard с финансами)
✅ Расписание (просмотр всех врачей)
✅ Пациенты
✅ Медкарта
✅ Финансы (все подразделы)
✅ Филиалы (управление)
✅ Сотрудники (управление, назначение ролей)
✅ Склад
✅ Аналитика (полная)
✅ Настройки (полный доступ)
```

### 2. Врач (Doctor)

**Клиническая работа:**

```
✅ Главная (упрощённый Dashboard)
✅ Расписание (своё расписание + просмотр коллег)
✅ Пациенты (все пациенты клиники)
✅ Зубная формула
✅ Медкарты (создание/редактирование)
✅ Услуги (просмотр прайса)
❌ Счета (только просмотр, без создания)
❌ Финансы (ограниченный доступ)
❌ Филиалы (нет доступа)
❌ Сотрудники (нет доступа)
❌ Склад (только просмотр остатков)
❌ Аналитика (базовая: свои записи)
✅ Настройки (только личный профиль)
```

### 3. Администратор (Admin)

**Операционное управление:**

```
✅ Главная (Dashboard с метриками)
✅ Расписание (управление записями всех врачей)
✅ Пациенты (полный доступ)
✅ Медкарты (просмотр)
✅ Счета (создание, отправка)
✅ Оплаты (приём платежей)
✅ Баланс пациентов
✅ Бонусы и акции
✅ Склад (управление)
❌ Филиалы (нет доступа)
❌ Сотрудники (нет доступа)
✅ Аналитика (операционная)
✅ Настройки (ограниченные)
```

## Детальная структура разделов

### 🏠 Главная (Dashboard)

**Подразделы:**
- Обзор дня (KPI, записи, задачи)
- Быстрые действия

**Роли:**
- Владелец: полные метрики + финансы
- Врач: своё расписание + свои пациенты
- Админ: операционные метрики

---

### 📅 Расписание

**Подразделы:**
- Календарь (день/неделя/месяц)
- Управление расписанием врачей
- Исключения (отпуска, больничные)

**Действия:**
- Создать запись
- Отменить/перенести
- Отметить статус (пришёл/не пришёл)

---

### 👥 Пациенты

**Подразделы:**
- Список пациентов (с фильтрами)
- Карточка пациента
- Импорт пациентов

**Карточка пациента:**
```
┌─────────────────────────────────────┐
│ Вкладки:                            │
│ • Общая информация                  │
│ • Контакты                          │
│ • История визитов                   │
│ • Зубная формула                    │
│ • Медкарты                          │
│ • Счета и платежи                   │
│ • Баланс и бонусы                   │
│ • Файлы (снимки)                    │
│ • Комментарии                       │
└─────────────────────────────────────┘
```

---

### 🦷 Зубная формула

**Функции:**
- Интерактивная 3D-модель (как в референсе)
- История изменений по зубам
- Привязка к медкартам

---

### 📋 Медкарты

**Подразделы:**
- История лечения
- Дневник врача
- Планы лечения
- Файлы (рентген, фото)

---

### 💰 Финансы

#### Счета
- Список счетов
- Создание счёта
- Статусы (ожидает/частично/оплачен)

#### Оплаты
- Приём платежей
- История платежей
- Кассовая смена (для администраторов)

#### Баланс пациентов
- Пополнение баланса
- Списание с баланса
- История операций

#### Бонусы и акции
- Бонусный баланс
- Правила начисления
- Акции и промо

---

### 🏢 Филиалы

**Функции:**
- Список филиалов
- Настройка филиала
- Переключатель активного филиала
- Кабинеты/кресла

---

### 👨‍⚕️ Сотрудники

**Подразделы:**
- Список пользователей
- Роли и права
- Специализации
- Доступ к филиалам

---

### 📦 Склад

**Подразделы:**
- Номенклатура материалов
- Остатки по филиалам
- Поступления
- Списания
- Контрагенты (поставщики)

---

### 📊 Аналитика

**Отчёты:**
- Выручка по периодам
- Загрузка врачей
- Популярные услуги
- Конверсия записей
- Пациенты без визитов
- Финансовые отчёты

---

### ⚙️ Настройки

**Разделы:**

#### Клиника
- Основные данные
- Часовой пояс, валюта
- Рабочие часы

#### Филиалы
- Управление филиалами

#### Сотрудники
- Добавить/удалить
- Роли и права

#### Услуги
- Категории услуг
- Прайс-лист
- Цены по филиалам

#### Виджет онлайн-записи
- Настройка API
- Доступные услуги
- Тема виджета

#### Документы и печать
- Шаблоны документов (договоры, согласия, справки)
- Пакеты документов (настройка готовых наборов)
- Настройки печати (принтеры, формат)
- Переменные автозаполнения

#### Интеграции
- Stripe/Paddle (биллинг)
- Email/SMS провайдеры
- API ключи

```
└─────────────────────────────────────────────────────┘
                        │
                        ▼
              ┌────────────────────┐
              │ ■ ЛЕН  Ленина   ✓ │ ← текущий
              │ ■ ЦНТ  Центр       │
              │ ■ ТЦ1  ТЦ Европа   │
              ├────────────────────┤
              │ ◉ Все филиалы      │
              └────────────────────┘
```

**Поведение:**
- При переключении филиала фильтруются:
  - Пациенты (можно выбрать "все")
  - Расписание
  - Склад
- Цвет филиала подсвечивает header

## Карта пациента

Вся клиническая информация, файлы, финансы пациента собраны в одной карте с вкладками.

**Детальная структура:** см. [PATIENT_CARD_STRUCTURE.md](PATIENT_CARD_STRUCTURE.md)

**Вкладки карты:**
1. 📋 Общие данные (персональная информация, контакты, аллергии)
2. 🏥 Клиническая карта (приёмы, диагнозы, планы лечения, дневник врача)
3. 🦷 Зубная формула (интерактивная карта зубов с историей)
4. 📁 Файлы и снимки (рентген, фото, документы)
5. 💰 Финансы (счета, оплаты, баланс, бонусы пациента)
6. 📊 История (единая хронология всех событий)
