# Структура медицинской карты пациента

> Единая карта пациента со всеми данными, разделённая на логические вкладки

---

## Концепция

**Один пациент = Одна карта = Все данные**

Вместо того чтобы прыгать между разными разделами (Медкарта → Зубная формула → Счета), весь функционал собран в одном месте.

```
Меню: "Пациенты" → Клик по пациенту → Открывается ПОЛНАЯ КАРТА
```

**Основные функции:**

1. **Быстрые пакеты** — печать 3-6 документов одной кнопкой (настраиваемые)
2. **Отдельные документы** — печать любого документа в любой момент
3. **Произвольный выбор** — любая комбинация документов + сохранение как пакет
4. **Архив** — все документы автоматически сохраняются с метаданными
5. **Повторная печать** — из архива за 1 клик
6. **Внешние документы** — загрузка сканов, направлений

- 📋 **Договорные:** договоры, согласия, анкеты

- 💰 **Финансовые:** счета, квитанции, акты
- 📎 **Внешние:** загруженные документы (сканы паспорта, направления)
