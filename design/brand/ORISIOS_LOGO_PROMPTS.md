- Clean circular outer shape (the letter O)

- The arc is slightly separated from the outer circle by negative space

- Single smooth stroke for all elements

Color: Deep teal gradient from #0D7377 to #14919B
The smile arc can have a subtle coral #FF6B6B hint

Style: Vector logo, flat design, minimal, Silicon Valley startup aesthetic
Background: Pure white
No text, just the symbol

Inspiration: Apple logo simplicity, Spotify icon energy, Figma logo cleverness

--ar 1:1 --v 6.1 --style raw

## Platform-Specific Tips

### Midjourney v6.1

- Use `--style raw` for logo work
- Add `--stylize 50` for less "artistic" interpretation
- `--no` negative prompts are crucial
- Generate in batches, iterate on best results
- Use `--chaos 0` for consistency

### DALL-E 3

- Start prompt with "Vector logo design" or "Minimalist logo"
- Be very explicit about "no text" if you want symbol only
- DALL-E tends to add text; emphasize "symbol only, no words"
- Specify "white background" explicitly
- "Flat design" helps avoid 3D renders

### Ideogram

- Excellent for text integration if needed
- Use "logo design" as first words
- Very literal interpretation; be precise
- Good for wordmark variations

### Leonardo AI

- Use "Logo Diffusion" or similar logo-trained models
- PhotoReal mode off for vector style
- Lower guidance scale (7-9) for logos
- Use negative prompt field

## Post-Generation Workflow

1. **Generate 20+ variations** of each concept
2. **Select top 5** from each batch
3. **Combine best elements** in Figma/Illustrator
4. **Vectorize** chosen direction
5. **Test at all sizes** (16px to 512px)
6. **Create color variations** (full, inverted, mono)
7. **Design wordmark pairing**
8. **Export final assets**
